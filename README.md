# GPX Track Viewer

Javascript-based GPX track viewer

## Installation and running

> `npm start`

## Usage

GPX track viewer can be used to display your GPX on a map, get the information about it like:  
- Length
- Elevation
- Slope
- Weather forecast including: 
    - Temperature
    - Wind Speed
    - Hourly Forecast
    - Precipitation
- Elevation and slope graph
